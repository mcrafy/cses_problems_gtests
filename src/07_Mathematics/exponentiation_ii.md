# Exponentiation Ii

- **Time limit:** 1.00 s
- **Memory limit:** 512 MB

Your task is to efficiently calculate values a^{b<sup>c</sup>} modulo 10<sup>9</sup>+7.

Note that in this task we assume that 0<sup>0</sup>=1.

# Input

The first input line has an integer n: the number of calculations.

After this, there are n lines, each containing three integers a, b and c.

# Output

Print each value a^{b<sup>c</sup>} modulo 10<sup>9</sup>+7.

# Constraints

- 1 ≤ n ≤ 10<sup>5</sup>
- 0 ≤ a,b,c ≤ 10<sup>9</sup>

# Example

Input:

```
3
3 7 1
15 2 2
3 4 5
```

Output:

```
2187
50625
763327764
```

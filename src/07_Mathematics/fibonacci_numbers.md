# Fibonacci Numbers

- **Time limit:** 1.00 s
- **Memory limit:** 512 MB

The Fibonacci numbers can be defined as follows:

- F\_0=0
- F\_1=1
- F\_n = F\_{n-2}+F\_{n-1}

Your task is to calculate the value of F\_n for a given n.

# Input

The only input line has an integer n.

# Output

Print the value of F\_n modulo 10<sup>9</sup>+7.

# Constraints

- 0 ≤ n ≤ 10^{18}

# Example

Input:

```
10
```

Output:

```
55
```

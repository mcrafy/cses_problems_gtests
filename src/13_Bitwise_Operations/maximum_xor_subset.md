# Maximum Xor Subset

- **Time limit:** 1.00 s
- **Memory limit:** 512 MB

Given an array of n integers, your task is to find the maximum xor sum of a subset.

# Input

The first line has an integer n: the size of the array.

The next line has n integers x\_1,x\_2,…,x\_n: the contents of the array.

# Output

Print one integer: the maximum xor sum of a subset.

# Constraints

- 1 ≤ n ≤ 2 · 10<sup>5</sup>
- 0 ≤ x\_i ≤ 10<sup>9</sup>

# Example

Input:

```
4
1 6 12 6
```

Output:

```
13
```

# Subsets With Fixed Average

- **Time limit:** 1.00 s
- **Memory limit:** 512 MB

You are given an array of n integers. Your task is to count the number of non-empty subsets of the given array with average equal to a.

# Input

The first line has two integers n and a: the size of the array and the target average.

The next line has n integers x\_1, x\_2,…, x\_n: the contents of the array.

# Output

Print one integer: the number of non-empty subsets with average equal to a, modulo 10<sup>9</sup> + 7.

# Constraints

- 1 ≤ n ≤ 500
- 1 ≤ a ≤ 500
- 1 ≤ x\_i ≤ 500

# Example

Input:

```
5 2
1 1 2 3 4
```

Output:

```
7
```

# Distinct Values Subarrays Ii

- **Time limit:** 1.00 s
- **Memory limit:** 512 MB

Given an array of n integers, your task is to calculate the number of subarrays that have at most k distinct values.

# Input

The first input line has two integers n and k.

The next line has n integers x\_1,x\_2,…,x\_n: the contents of the array.

# Output

Print one integer: the number of subarrays.

# Constraints

- 1 ≤ k ≤ n ≤ 2 · 10<sup>5</sup>
- 1 ≤ x\_i ≤ 10<sup>9</sup>

# Example

Input:

```
5 2
1 2 3 1 1
```

Output:

```
10
```

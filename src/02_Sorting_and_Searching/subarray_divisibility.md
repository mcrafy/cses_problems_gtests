# Subarray Divisibility

- **Time limit:** 1.00 s
- **Memory limit:** 512 MB

Given an array of n integers, your task is to count the number of subarrays where the sum of values is divisible by n.

# Input

The first input line has an integer n: the size of the array.

The next line has n integers a\_1,a\_2,…,a\_n: the contents of the array.

# Output

Print one integer: the required number of subarrays.

# Constraints

- 1 ≤ n ≤ 2 · 10<sup>5</sup>
- -10<sup>9</sup> ≤ a\_i ≤ 10<sup>9</sup>

# Example

Input:

```
5
3 1 2 7 4
```

Output:

```
1
```
